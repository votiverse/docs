# Votiverse Documentation

Documentation, case studies, articles, and papers for [Votiverse](https://github.com/votiverse/votiverse) — a configurable governance engine for democratic decision-making.

---

## Case Studies

Real-world scenarios demonstrating Votiverse in action, with screenshots from the live application.

- **[Maple Heights Condo Board](case-studies/maple-heights-condo-board/)** — How a 24-unit condominium uses Votiverse to make transparent, accountable decisions about their shared building. Covers group creation, admission control, proposals, community notes, voting, and results.

## Guides

Practical guides for working with Votiverse.

- **[Screenshot Workflow](guides/screenshot-workflow.md)** — How to create reproducible, high-quality screenshots from the Votiverse UI using Playwright.
